What is AmurotOpenSource?

AmurotOpenSource is a training repository where you learn how to contribute to open source projects by actually doing it. Fork the repo, create your page, fix bugs, and collaborate with others — all in a safe, beginner-friendly environment.

★

Real Workflow

Work with Git branches, pull requests, and code review — just like professional developers.

⚙

Fix Real Bugs

This website contains intentional bugs. Find them, fix them, and open a pull request.

◆

Your Own Page

Every contributor gets a personal page at /people/your-username.

☍

Collaborate

Review other contributors' pull requests. Leave comments. Help each other learn.

How It Works

  1. 1

    Fork & Clone

    Fork this repository and clone it to your computer.

  2. 2

    Create Your Page

    Copy the template, fill in your details, and commit your work.

  3. 3

    Open a Pull Request

    Push your branch and open a PR. A maintainer will review it.

  4. 4

    Go Further

    Fix bugs, improve docs, build features. Level up your skills.
